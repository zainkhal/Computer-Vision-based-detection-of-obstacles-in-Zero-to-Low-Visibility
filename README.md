Dataset Link : https://app.roboflow.com/aaz/low-visibility-final-bg938/train
Dataset Description:
This dataset was created and uploaded on Roboflow and consists of three object classes: Car, Person, and Bike. It is prepared for object detection tasks and can be used for training models in real-time applications. The dataset is structured to support computer vision projects such as traffic monitoring, surveillance, and smart transportation systems.
